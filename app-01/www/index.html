Task-01

1. Развернуть проетк(github) или gitlab

2. Написать role на ansible который бы выполнял:

  1. обновление системы
  2. создание пользователя с sudo правами и создаными ssh ключами
  3. установка docker(версии 19.03.06)
  4. инициализация swarm стека:
    • в стеке должна быть описана overlay сеть с явным указанием mtu
    • nginx
    • mysql
    • php-fpm(wordpress,sympfony...) - можно взять и не php-fpm, любую технологию(flask например)
    • код минимум с одной html страничкой для проверки результа
    • volumes для mysql и php-fpm(чтобы данные после перезагрузки не удалялись)
    • docker config и docker secret для сервиса nginx
    • Создать самоподписные сертификаты и nginx конциги(желательно в ansible- но можно полдожить и в репу и просто перекидывать).
    • Наружу должны быть открыты только 80 и 443 порт

3. Предоставить Код на gitlab(github)

4. Дополнительно:

  1. Связать бд с php-fpm и вывести какие-то данные с Бд.
  2. Написать ansible playbook котрый бы обновлял сертификаты и конфиги на сервере.
  3. Если не получиться подкинуть docker config и docker secret можно простым пробрасыванием volumes в nginx

Результаты:

  1. Стандартный вывод info.php

  2. Генератор паролей на PHP

  3. Проверка соединения с базой данных

  4. Вывод информации с базы данных WORLD, импортированой для теста
